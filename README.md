## ‚ú® Features

* **Single-Page Design:** A clean, modern, one-page layout.
* **Responsive & Mobile-Friendly:** Includes a hamburger menu and responsive grid layouts that look great on all devices.
* **Smooth Scrolling:** Smooth, animated scrolling for navigation links.
* **Dynamic Animations:** Subtle fade-in-on-scroll animations for a modern feel.
* **Auto-Hiding Navbar:** The navigation bar hides on scroll-down and reappears on scroll-up.
* **Functional Contact Form:** An AJAX-powered contact form integrated with Formspree that sends emails directly.
* **Icon Library:** Uses Font Awesome for high-quality social media icons.

This website is built with clean, modern, front-end technologies:
